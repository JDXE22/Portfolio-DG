# Portfolio - DG

A modern, minimalist, and high-performance personal portfolio built with [Astro](https://astro.build/).

## 🚀 Features

- **Blazing Fast**: Built with Astro for optimized performance and minimal JavaScript.
- **Modern Tech Stack**: Leveraging the latest web technologies.
- **Responsive Design**: Fully responsive and mobile-friendly.
- **Clean Codebase**: Organized and easy to maintain.

## 🛠️ Tech Stack

- **Framework**: [Astro](https://astro.build/)
- **Styling**: CSS (Vanilla)
- **Package Manager**: [pnpm](https://pnpm.io/)

## 📂 Project Structure

```text
/
├── public/          # Static assets like favicon, robots.txt
├── src/
│   ├── assets/      # Images and global styles
│   ├── components/  # Reusable UI components
│   ├── layouts/     # Page layouts
│   └── pages/       # Project pages (routing)
├── astro.config.mjs # Astro configuration
└── package.json     # Project dependencies and scripts
```

## 🧞 Commands

All commands are run from the root of the project:

| Command        | Action                                      |
| :------------- | :------------------------------------------ |
| `pnpm install` | Installs dependencies                       |
| `pnpm dev`     | Starts local dev server at `localhost:4321` |
| `pnpm build`   | Build your production site to `./dist/`     |
| `pnpm preview` | Preview your build locally                  |
